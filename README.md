## 在线邀请函

本项目包含一个功能完整的在线邀请函网页，支持自动邮箱绑定功能。

### 快速开始

1. 打开 `invitation.html` 文件即可使用
2. 支持通过URL参数自动填充邮箱：`invitation.html?email=your@email.com`

### 功能特性

- ✉️ **自动邮箱绑定** - 通过URL参数自动获取和填充邮箱地址
- ✅ **实时表单验证** - 即时验证邮箱格式和必填项
- 💾 **本地数据存储** - 使用localStorage保存绑定记录
- 📱 **响应式设计** - 完美支持手机和平板设备
- 🎨 **优雅界面** - 现代化渐变设计，流畅动画效果

详细使用说明请查看 [INVITATION_README.md](INVITATION_README.md)
